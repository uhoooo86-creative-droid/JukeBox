사용 방법
01
아래에서 앨범 정보와 트랙 목록을 입력합니다.
음원 파일명은 실제 업로드할 파일명과 정확히 일치해야 합니다.
02
JSON 생성 버튼으로 tracks.json 내용을 생성합니다.
03
생성된 JSON을 복사해서 tracks.json 파일에 덮어쓰고, 음원 파일들은 audio/ 폴더에 넣은 뒤 GitHub에 push합니다.
04
GitHub Pages가 자동으로 반영합니다. 🎉
